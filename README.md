# AdvancedBanX For Velocity

AdvancedBanX or ABX for short never supported Velocity 
even if it is one of the biggest plugins for bans in history but not updated or with no Velocity Support.
This Velocity Plugin fixes 1 of those problems. No one uses bungeecord

## Building
 `
./gradlew shadowJar
 `
## How to use
Add the .jar (jarfile) to you server plugin directory and restart the server.
To know how to use it go [here](https://modrinth.com/plugin/advancedbanx).
It is just the BungeeCord Plugin with velocity support.
